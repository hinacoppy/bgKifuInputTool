Backgammon Kifu Input Tool

◆このアプリは何？

バックギャモンの棋譜を入力するためのツールです。
eXtream Gammon(XG)にも棋譜を入力する機能はありますが、棋譜入力に特化して、かつXGの棋譜入力機能より使いやすいツールとして提供することを目的として開発しました。
マウス操作ができるパソコンでのみ稼働させることができます。スマホやタブレットなどのタッチデバイスでは動きません。
犬さんにツール開発のヒントをいただきました。＞Thanks to 犬さん

◆何ができるの？

ビデオで撮影した動画を再生しながら、その棋譜を入力します。
入力した棋譜はMAT形式の棋譜データとして出力することができます。
出力したMAT形式のファイルはXGやGnuBGに読み込むことができます。解析などはそちらのツールで実行させてください。

◆どうやって使うの？

(1) 棋譜入力準備

・対戦者の名前、マッチポイント数、イベント名、イベント日を入力します。
　→これらの情報は棋譜ファイルにコメントとして記述されます。
・を押下し、棋譜入力を始めます。
・[Action Panel] はドラッグ＆ドロップで動かすことができます。使いやすい場所に移動してください。

(2) 棋譜入力

・ダイス一覧が表示され、出目を選択します。
　→オープニングロールでは先攻・後攻を指定することができます。
　→この画面ではキーボードの 123456 と d の入力を受付けます。
・マウス操作でチェッカーを動かすことができます。
　→チェッカーはドラッグ＆ドロップとクリックで操作することができます。
　→クリック操作では大きい目のダイスから使います。
・[Done] ボタンを押下するとチェッカー操作を終えます。
　→ダイス部分の左クリックで [Done] と同じ機能が動きます。
・[Undo] ボタンを押下するとチェッカー操作をやり直すことができます。
　→ダイス部分の右クリックで [Undo] と同じ機能が動きます。
・[Done][Undo] の画面ではキーボードの Enter, Space と Esc を受付けます。
　→ Enter, Space は [Done] の機能として、Esc は [Undo] の機能として動きます。

(3) ダブリング操作

・ダイス一覧画面の [Double] を押下するとダブル提案です。
・ダブルを受けた側は [Take] [Pass] のいずれかを押下します。
　→[Take] のときは棋譜入力を継続します。
　→[Pass] のときはゲーム終了画面が表示されます。
　→[Take][Pass] の画面ではキーボードの t と p の入力を受付けます。

(4) リザイン

・ダイス一覧画面の [Resign] を押下するとリザイン(負け宣言)です。
　→ゲーム終了画面が表示されます。
　→得点は自動的に計算されます。
　→シングル、ギャモン、バックギャモンの判定は単純なロジックで実装しています。

(5) イリーガルムーブ

・[Allow input of illegal moves] にチェックするとイリーガルムーブの入力ができるようになります。
　→掴んだチェッカーより前方のマスであればどこにでもムーブすることができます。
　→この状態ではチェッカーをドラッグ＆ドロップで操作します。

(6) 手順戻し

・ を押下すると手順を戻すことができます。
　→数手順前の棋譜入力ミスに気付いたときに使う機能です。

(7) 棋譜ダウンロード

・ を押下し棋譜をダウンロードします。
　→ダウンロードしたファイルはMATフォーマットのテキストファイルです。
　→このファイルは eXtream Gammon や GnuBG などの解析ツールに読み込ませることができます。
　→解析はこれらのツールで実行してください。

◆ボタンとその機能

設定をクリアして新しいマッチの棋譜入力を始める
入力した棋譜の手順を戻す
ボードの左右を反転する
棋譜をダウンロードする
この説明書を開く
ホーム画面に戻る

◆分かっている（仕様という名の）問題点

・目を余らせて動かしたときに、次のコマがクリックで動かせないことがある
　ex. ベアオフで、54の目で4ptを先にクリックでベアオフすると、5ptのコマはクリックでベアオフできない
・ダイス組み合わせの位置にD&Dで動かしたときは、次のクリックムーブが正しく動かないことがある
